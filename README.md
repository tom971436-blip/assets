# assets

A repository for storing images and other assets.

## Structure

```
/images   # Image hosting
/files    # Other assets (json, docs, etc.)
```

## Usage

### Image (via CDN)

```
https://cdn.jsdelivr.net/gh/tom971436-blip/assets@main/images/<path>
```

## Notes

* This repo is public for CDN access.
* Directory structure is kept simple on purpose.
